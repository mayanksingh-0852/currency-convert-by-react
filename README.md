# 💱 Currency Converter (React + Custom Hooks)

A simple and responsive **Currency Converter Web Application** built using **React.js** and **Custom Hooks**.  
This app fetches real-time exchange rates and allows users to convert currencies instantly.

---

## 🚀 Features

- 💱 Convert one currency to another
- 🔁 Swap currencies instantly
- 🌍 Multiple currency options
- ⚡ Real-time exchange rates using API
- 🎨 Clean UI built with Tailwind CSS
- ♻️ Reusable components

---

## 🧠 What I Learned

- Creating and using **Custom Hooks in React**
- Fetching and handling **API data**
- Managing state using **useState**
- Building reusable components
- Implementing dynamic currency conversion logic

---

## 🛠 Tech Stack

- React.js
- JavaScript (ES6)
- Tailwind CSS
- Custom Hooks
- API Integration
